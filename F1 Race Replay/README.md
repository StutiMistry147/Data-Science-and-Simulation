# F1 Race Relay
## Overview
- F1 Race Relay is a Formula 1 race visualization and data analysis tool built with Python.
- It transforms raw F1 timing data into interactive visualizations, comprehensive dashboards, and detailed race analytics through an elegant, modern interface.

## Tech
1. Language: Python
2. Data Source : FastF1
3. GUI Framework : PySide6
4. Data Processing : Pandas	

## Execution
1. Clone or download the project
```
git clone https://github.com/yourusername/f1-race-relay.git
cd f1-race-relay
```
2. Install dependencies
`pip install -r requirements.txt`
3. Verify installation
`python launch.py`

Method 1: Use the launcher (recommended)
`python launch.py`

Method 2: Direct GUI launch
`python modern_gui.py`

Method 3: Generate visualizations only
`python modern_plots.py`
